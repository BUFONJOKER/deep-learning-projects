# TensorFlow ANN

Small, focused TensorFlow ANN examples for classification tasks, designed to be easy to run and modify.

This folder contains Artificial Neural Network (ANN) projects implemented with TensorFlow.

## Contents

- **01-student-depression-ann.ipynb**: ANN model for student depression classification.

## Usage

Open the notebook in Jupyter or VS Code and run cells from top to bottom.

## Requirements

- Python 3.9+
- TensorFlow
- Jupyter (or VS Code with Jupyter extension)
